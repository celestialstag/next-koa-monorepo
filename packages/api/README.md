### Starting

#### **Step 1 - Run the start script**

### **Step 1 - Copy `.env.example` to `.env`**

```bash
cp .env.example .env
```

### **Step 2 - Populate environment variables**

```bash
vim .env
```

#### **Step 3 - Run the start script**

```bash
yarn start:api
```

## Developing

#### **Step 1 - Build common libraries**

```bash
yarn build:libs
```

#### **Step 1 - Watch the source files**

```bash
yarn serve:api
```

### Running

```bash
yarn start:api
```

### Testing
